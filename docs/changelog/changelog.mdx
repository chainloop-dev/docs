## v0.8.58 - 2023-01-08

- UX improvement: Allow the creation/import of a contract during workflow creation

```bash
# Before you could reference an existing contract by it's UUID
chainloop workflow create --name release --project skynet --contract [UUID]
# Now, you can also provide an existing contract file local path
chainloop workflow create --name release --project skynet --contract ./skynet.contract.yaml
# ... or URL
chainloop workflow create --name release --project skynet --contract https://skynet.org/contract.yaml
```
